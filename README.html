Restaurant Management System

Welcome to the Restaurant Management System, a web application built using PHP, HTML, CSS, JavaScript, Bootstrap, and powered by a MySQL database. This system is designed to streamline the process of managing reservations and menus for a restaurant. The project includes two main interfaces: one for clients to make reservations and view menus, and another for administrators to manage menus, dishes, reservations, and their profiles.

Features

  • Client Interface:
    • Make Reservations: Clients can easily make reservations for a specific date and time.
    • View Menus: Clients can explore the restaurant's menu with detailed descriptions of each dish.
  • Administrator Interface:
    • Edit Menu: Admins have the ability to add, update, or remove dishes from the menu.
    • Manage Reservations: View and manage reservations made by clients.
    • Profile Management: Admins can modify their profiles, including updating passwords.
